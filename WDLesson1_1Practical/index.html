Web Technologies

HTML

HTML is the abbreviation for HyperText Markup Language. It brings in the element to the web page. It is not a programming language.

CSS

CSS is the abbreviation for cascading styles sheet. It allows one to style/decorate the element. It's used to make the webpage pretty.

JavaScript

JavaScript is a programming language. It allows one to control the behavior of the element. We are going to use Javascript this year for web development.
